## 📝 LESSONS LEARNED

### For Reporting Architecture
1. **Consistent Pagination:** `as_of` pattern works well for large datasets
2. **Database Views:** Complex aggregations cleaner in SQL views
3. **Snapshot Reads:** Consistent reads prevent data inconsistencies
4. **Role-Based Access:** Different reports for different roles

### For Export Systems
1. **Scheduled Exports:** Powerful for recurring reports
2. **Format Flexibility:** Multiple export formats needed
3. **Async Processing:** Large exports should be background jobs
4. **Retention:** Exported files need retention policies
